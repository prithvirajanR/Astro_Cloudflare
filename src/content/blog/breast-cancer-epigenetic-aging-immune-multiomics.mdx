This project asks whether aging-related methylation signals and immune context can add useful information for breast cancer subtype classification. The analysis focuses on TCGA BRCA samples and PAM50 subtype labels, especially Basal, Luminal A, and Luminal B.

Instead of treating subtype prediction as a single-table machine-learning exercise, the workflow builds multiple data layers: clinical metadata, RNA expression, DNA methylation, epigenetic age acceleration, immune scores, and subtype labels.

## Research question

Do epigenetic age acceleration and immune deconvolution features provide interpretable signal for distinguishing PAM50 breast cancer subtypes?

## Methods

DNA methylation clocks estimate biological age from methylation patterns. The relevant cancer feature is not only predicted age, but the difference between methylation age and chronological age. That delta-age value can indicate whether a tumor sample appears biologically older or younger than expected.

The statistical report compared Horvath, Hannum, and PhenoAge clocks across subtype groups. The main pattern in the report was subtype-specific: Luminal tumors showed positive age acceleration across clocks, while Basal tumors clustered closer to zero or negative delta-age. Hannum had the strongest overall fit in the downstream linear model, with the report noting an approximate R-squared around 0.30.

## Immune context

The immune analysis connected delta-age patterns to tumor immune composition. The annotation report compared immune cell scores against delta Hannum age and subtype labels, then brought those features into model interpretation.

Two findings made the project more biologically interesting:

- Resting mast cells and M0 macrophages appeared as strong delta-age correlates, suggesting an innate or wound-healing-like immune profile in age-accelerated tumors.
- Follicular helper T cells and macrophage patterns helped separate Luminal and Basal signals, while SHAP-style feature importance highlighted immune features that also mattered to subtype decisions.

Those results make the project less about classifier accuracy alone and more about whether the predictive features are biologically interpretable.

## Outputs

The notebooks cover the full workflow: metadata cleaning, RNA preprocessing, methylation preprocessing, delta-age calculation, immune score integration, multiclass classification, and annotation. The repo also includes two report artifacts: an epigenetic age statistical analysis PDF and an annotation PDF.

The result can be read as both a reproducible analysis and a biological interpretation exercise. The modeling task is subtype classification, but the value is in connecting aging, immune state, and breast cancer subtype biology.

## Limitations and next steps

The next step would be to harden the workflow for reuse: replace absolute paths in the R Markdown reports, add a single configuration file for data locations, and make the model evaluation outputs easier to regenerate from a clean checkout.

The current result should be interpreted as an applied multi-omics analysis, not a clinical classifier. A stronger version would include cleaner regeneration instructions, more explicit train/test reporting, and sensitivity analysis across feature groups.
