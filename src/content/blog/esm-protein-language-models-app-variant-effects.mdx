This project evaluates whether protein language model scores can recover experimentally measured mutation effects in human Amyloid Precursor Protein (APP/A4_HUMAN). The goal is not to claim that a model understands APP biology in a mechanistic sense, but to test how far sequence-derived scores can be pushed against a real deep-mutational scanning benchmark.

The dataset is `A4_HUMAN_Seuma_2021` from ProteinGym, with approximately 14,483 APP variants. Its mix of single and double mutants makes it useful for comparing both direct variant-effect prediction and epistasis-related behavior.

## Research question

Can zero-shot ESM scores provide a useful ranking of APP variants, and do different scoring formulations behave differently on single-mutant, double-mutant, and synthetic-landscape analyses?

## Methods

The repository compares ESM-1v and ESM-2 models using several scoring strategies:

- masked log-likelihood ratio
- pseudo-log-likelihood
- embedding distance from wild type
- entropy-weighted masked scores
- ensemble-style masked scores
- mutant marginal probability

The scoring function is treated as part of the experimental design. Two models with the same architecture family can produce different rankings depending on whether the analysis uses masked probabilities, whole-sequence pseudo-likelihood, or embedding geometry.

## Evidence

The metrics artifacts report Spearman correlations against experimental DMS scores across the APP variant set. The strongest runs were in the low-to-mid 0.4 range, which is plausible for zero-shot protein variant-effect prediction and should be interpreted as partial ranking signal rather than direct functional explanation.

Examples from the results file:

- ESM2 650M EnsembleMLLR reached rho = 0.4234 with P100 = 13%.
- ESM2 650M PLL reached rho = 0.4253 with P100 = 10%.
- The top predictions.csv summary reached rho = 0.4265 with P100 = 11%.
- ESM2 650M EntropyMLLR reached rho = 0.4196.

The main pattern is methodological rather than simply architectural: larger ESM-2 models and stronger scoring formulations often improved performance, but complexity did not guarantee a better result.

## Beyond a leaderboard

The project also includes epistasis analysis and synthetic fitness-landscape exploration. This matters because double-mutant behavior is where additive assumptions can fail, and where a language model score may capture useful context without being a full causal model of function.

The repository preserves figures, CSV outputs, HPC scripts, and a compiled report. That makes the result auditable: the work is not only a set of console outputs, but a collection of artifacts that can be reviewed after the run.

## Limitations and next steps

These results should not be overread as proof of mechanistic understanding. A moderate rank correlation can still be useful for prioritization, but it does not replace experimental validation or structural interpretation.

The next improvement would be a cleaner ablation table separating model scale, scoring formulation, variant class, and evaluation metric. That would make it easier to distinguish real model signal from scoring artifacts.
